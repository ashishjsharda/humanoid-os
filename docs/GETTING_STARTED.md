# Getting Started

## Install
```bash
git clone https://github.com/ashishjsharda/humanoid-os.git
cd humanoid-os
pip install -r requirements.txt
```

## Run Demo
```bash
python examples/kinematics_demo.py
```

That's it! The robot performs a zero-gravity actuator system check

## What You Get

- Real-time control (1kHz)
- ZMP balance
- Walking gaits
- PyBullet simulation

See [ARCHITECTURE.md](ARCHITECTURE.md) for details.
